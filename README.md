## ssl-check

```bash
Usage:
	ssl-check [OPTION] [ARG]

Options:
	-d, --domain            指定域名
	-l，--list              指定域名列表
	-t, --timeout           指定超时时间
	-T, --timezone          指定时区
	-p, --par               指定并发数
	-L, --log               生成日志文件
	-v, --version           查看版本信息
	-h，--help              查看帮助
		
Example:
	ssl-check -Ld example.com
```
